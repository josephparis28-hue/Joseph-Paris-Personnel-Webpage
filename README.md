# Project 01: Personal Profolio Webpage

---

## Project Purpose

The purpose of this project is to apply the project management principle of **scoping** by clearly defining the goals, deliverables, and limitations of a technical task. This project focuses on planning and developing a single page personal webpage using semantic HTML5.[^1]

---

## Project Description

This project involves the development of a single page personal profolio website built with valid HTML5. The web page will be used to give background information about who I am. The primary emphasis is on semantic structure, accessibility, and clarity.

---

## Desired Results for Week 3

- A clearly defined scope statement completed before development begins [^1]
- A valid HTML5 document that uses semantic elements correctly
- A professional single‑page personal webpage that introduces me and my interests  
- Demonstrated understanding of how project scoping improves focus and efficiency  
- Front load all necessary work for upcoming deliverables [^1]
- First deliverable due April 5, 2026 by 11:59pm.
- Second deliverable due Apr 19 by 11:59pm.
- Final deliverable due May 8 by 11:59pm.

---

## Deliverables for Week 3

- `scope.md` — a formal project scope statement written in Markdown  
- `index.html` — a single‑page semantic HTML document  

---

## Exclusions for Week 3

The following items are explicitly out of scope for this project:  

- JavaScript functionality  
- User authentication  
- Multi‑page navigation  
- External libraries  
- Style file for formatting (Only prepared for when style.css will be needed)

---

## Constraints

- My Webpage project must meet all minimum assignment requirements  
- The final academic submission must be a single .zip file containing only:  
  - `scope.md`  
  - `index.html`  

---

## Acceptance Criteria

The project will be considered complete when:  

- `scope.md` clearly defines purpose, description, desired results, and exclusions[^1]  
- `index.html` is a valid HTML document and includes:  
  - Title Element in Head
  - H1 Element with Name
  - H2 Element for Professional Interest
  - Paragraph for About Me
  - Paragraph for Professional Interests
  - Paragraph for Goals
  - An Unordered List for Technical Skills
  - An Unordered List for Personal Interests
- Section for Projects
- Section for Contact
- Hyperlink to My GitHup Profile
- The webpage containts proper syntax
- Front load all CCS fromatting in index.html for style.css
